# 贡献者公约

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺：参与本项目的每个人都不会受到骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向身份。

我们承诺以促进开放、包容、多元化、健康社区的方式行动和交流。

## 我们的标准

正面行为示例：

- 对他人展示同理心和善意
- 尊重不同意见和观点
- 给予建设性反馈，也优雅地接受反馈
- 承认错误、道歉，并从中学习
- 关注的不仅是个人，更是整个社区的利益

不可接受的行为：

- 使用性化语言或意象
- 人身攻击、恶意挑衅、侮辱性评论
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息
- 其他违反职业道德的行为

## 执行责任

社区领导者负责澄清和执行本公约标准，并将对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

## 适用范围

本公约适用于所有项目空间，也适用于个人在公共空间代表项目或社区时的行为。

## 执行

辱骂、骚扰或其他不可接受的行为可通过联系项目维护者举报。所有投诉都将得到及时和公正的审查和处理。
